# Prompt Library
This repository contains the new prompt library for Roo code agents.

## Structure
- `prompts/`: Contains all the prompt files.
- `docs/`: Contains documentation related to the prompts and usage.

## Usage
To use the prompt library, follow these steps:
1. Clone this repository.
2. Use the prompts in your Roo code agents.
